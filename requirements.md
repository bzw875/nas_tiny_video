# NAS 视频文件管理系统 - 需求文档

## 1. 项目概述

### 1.1 背景
NAS (192.168.1.17) 上存储器大量视频文件，需要将非相册视频导入数据库，实现便捷的检索、标签管理和网页播放功能。

### 1.2 目标
- 爬取 SMB 指定目录下的所有视频文件
- 跳过图片等其他文件类型
- 将视频元数据存储到数据库
- 提供检索、打标签、网页播放功能

## 2. 功能需求

### 2.1 SMB 爬虫模块
| 需求ID | 需求描述 | 优先级 |
|--------|----------|--------|
| F2.1.1 | 连接 SMB 服务器 (192.168.1.17) | P0 |
| F2.1.2 | 递归遍历指定目录 | P0 |
| F2.1.3 | 识别视频文件类型 (mp4, mkv, avi, mov, wmv, flv, webm 等) | P0 |
| F2.1.4 | 跳过非视频文件 (图片、文档等) | P0 |
| F2.1.5 | 提取视频元数据 (文件名、大小、修改时间、路径) | P0 |
| F2.1.6 | 增量爬取 (只处理新文件或修改的文件) | P1 |

### 2.2 数据存储模块
| 需求ID | 需求描述 | 优先级 |
|--------|----------|--------|
| F2.2.1 | 设计视频信息数据库表结构 | P0 |
| F2.2.2 | 支持视频信息增删改查 | P0 |
| F2.2.3 | 支持标签管理 | P1 |
| F2.2.4 | 支持全文搜索 | P1 |

### 2.3 Web 管理界面
| 需求ID | 需求描述 | 优先级 |
|--------|----------|--------|
| F2.3.1 | 视频列表展示 (分页) | P0 |
| F2.3.2 | 视频搜索功能 (按文件名) | P1 |
| F2.3.3 | 标签管理 (添加/删除/编辑标签) | P1 |
| F2.3.4 | 视频播放器 (HTML5 Video) | P0 |
| F2.3.5 | SMB 流式传输支持 | P0 |

### 2.4 管理功能
| 需求ID | 需求描述 | 优先级 |
|--------|----------|--------|
| F2.4.1 | 手动触发爬取 | P0 |
| F2.4.2 | 爬取进度显示 | P1 |
| F2.4.3 | 爬取日志查看 | P2 |

## 3. 技术选型建议

### 3.1 后端
- **运行时**: Node.js
- **SMB 客户端**: `smb2` 或 `@smb2/smb2`
- **Web 框架**: Express.js
- **数据库**: SQLite (轻量级)
- **ORM**: TypeORM

### 3.2 前端
- **框架**: React
- **UI 组件**:  Ant Design
- **视频播放器**: Video.js 或原生 HTML5 Video

## 4. 非功能需求

### 4.1 性能
- 单次爬取处理 10,000+ 文件不卡顿
- 视频列表加载时间 < 1 秒
- 视频播放延迟 < 2 秒

### 4.2 可靠性
- SMB 连接失败自动重试
- 爬取中断可断点续传
- 数据库操作事务支持

### 4.3 可扩展性
- 支持配置多个 SMB 源
- 支持插件式扩展视频类型识别

## 5. 数据库设计

### 5.1 videos 表
| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | INTEGER | 主键 |
| filename | VARCHAR | 文件名 |
| path | VARCHAR | 完整路径 |
| size | BIGINT | 文件大小 (字节) |
| modified_at | TIMESTAMP | 文件修改时间 |
| created_at | TIMESTAMP | 记录创建时间 |
| smb_host | VARCHAR | SMB 主机地址 |

### 5.2 tags 表
| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | INTEGER | 主键 |
| name | VARCHAR | 标签名称 |
| color | VARCHAR | 标签颜色 |

### 5.3 video_tags 表 (关联表)
| 字段名 | 类型 | 说明 |
|--------|------|------|
| video_id | INTEGER | 视频 ID |
| tag_id | INTEGER | 标签 ID |

## 6. 实施阶段

### 阶段一: 基础爬虫 (P0)
- SMB 连接和目录遍历
- 视频文件识别和元数据提取
- 数据库存储

### 阶段二: Web 播放 (P0)
- 视频列表展示
- SMB 流式传输
- 基础播放器

### 阶段三: 高级功能 (P1)
- 搜索功能
- 标签管理
- 增量爬取

### 阶段四: 优化 (P2)
- 爬取日志
- 多源支持
- 性能优化
